# dade-neocities

A period-correct 1998 Geocities-style homepage, hand-built for Neocities.

## Assets

GifCities and other Wayback-sourced Geocities mirrors were treated as **license-unclear** for this pass, so nothing was pulled from there. Everything below is either **Wikimedia Commons** (file-page license), **Public Domain** artwork, or **original CC0** raster/MIDI generated locally with Pillow for this repository.
